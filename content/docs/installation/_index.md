## Linux

{{< tabs items="Sheepitmd,Sheepitm,Sheepitm-web" >}}

{{< tab >}} - Clone the [repo](https://github.com/KoolKid6958/sheepitmd) with `git clone https://github.com/KoolKid6958/sheepitmd.git` - Run `cargo run -r -- -h` to see avaiable options.
{{< /tab >}}

{{< tab >}} - Clone the [repo](https://github.com/KoolKid6958/sheepitm) with `git clone https://github.com/KoolKid6958/sheepitm.git` - Run either `cargo gui -r`, or `cargo tui -r` depending on if you want the gui or tui.
